## 📖 完整练习题库

### 难度1（基础）

**1. 输出九九乘法表**
```go
// 完成这个程序
func main() {
    // 使用两层for循环输出9*9乘法表
}
```
**答案与讲解：**
```go
for i := 1; i <= 9; i++ {
    for j := 1; j <= i; j++ {
        fmt.Printf("%d*%d=%d ", j, i, j*i)
    }
    fmt.Println()
}
```

**2. 计算斐波那契数列**
```go
func fibonacci(n int) int {
    // 实现递归版本
}
func main() {
    for i := 0; i < 10; i++ {
        fmt.Println(fibonacci(i))
    }
}
```
**答案**：
```go
func fibonacci(n int) int {
    if n <= 1 {
        return n
    }
    return fibonacci(n-1) + fibonacci(n-2)
}
```

### 难度2（进阶）

**3. 并发计算多个数值的平方**
```go
func main() {
    results := make(chan int, 100)
    var wg sync.WaitGroup
    for i := 1; i <= 100; i++ {
        wg.Add(1)
        go func(n int) {
            defer wg.Done()
            results <- n * n
        }(i)
    }
    wg.Wait()
    close(results)
    for result := range results {
        fmt.Println(result)
    }
}
```
